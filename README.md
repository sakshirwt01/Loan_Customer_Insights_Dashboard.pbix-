# 📊 Loan Customers Insights Dashboard (Power BI)

## 📌 Overview

This project is a **Power BI dashboard** designed to analyze loan customer data and extract meaningful business insights. It provides a clear visualization of customer demographics, income distribution, loan amounts, and repayment trends.

## 🎯 Objectives

* Analyze customer income patterns
* Understand loan distribution and terms
* Identify demographic trends (gender, marital status, area)
* Support data-driven decision-making

## 🛠️ Tools & Technologies

* Power BI
* Data Visualization
* Data Cleaning & Transformation (Power Query)
* DAX (Data Analysis Expressions)

## 📈 Key Insights

* Majority of applicants fall within the **200K–400K income range**
* Most loans are issued for **241–360 months**
* Higher proportion of customers are **married**
* Semi-urban areas show strong loan distribution
* Male applicants dominate the dataset

## 🚀 How to Use

1. Download the `.pbix` file
2. Open it using Power BI Desktop
3. Explore interactive visuals and filters
